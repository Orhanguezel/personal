Antalya Döner & Pizzeria

This web application streamlines the restaurant management process for Antalya Döner & Pizzeria. The project includes online ordering, menu browsing, and order tracking. It allows customers to place orders for delivery or in-store pickup, customize their meals, and create user accounts for personalized experiences. Admins can manage orders and view analytics to improve operational efficiency.

Technologies Used:

  • Frontend: React, Bootstrap
  • Backend: Node.js, Express.js
  • Database: MongoDB

Key Features:

  • Real-time order management and updates
  • User registration and personalized ordering
  • Admin dashboard with order analytics and management
  • Mobile-responsive design for easy access on the go

MissEt

MissEt is a web-based project designed to offer a seamless solution for meat lovers to explore premium meat products and make table reservations at our partner restaurants. While our online reservation system is still under development, the platform currently offers a detailed product browsing experience where users can view product specifications and choose their desired items for in-store purchases.

Technologies Used:

  • HTML, CSS, JavaScript

Key Features:

  • Detailed product catalog for different cuts and types of meat
  • In-progress: Online reservation system for quick and easy table bookings
  • Fully responsive design for both desktop and mobile users

Kuhlturm

This project was developed for Ensotek, a refrigeration company specializing in industrial cooling solutions. The website provides a comprehensive overview of their products and services, focusing on sectors like chemical industry and power generation.

Technologies Used:

  • HTML, CSS, JavaScript, Bootstrap

The website was designed with a modern and professional interface to convey the technical precision of the company’s work. The design is fully responsive, ensuring compatibility across devices, and includes detailed service information to assist clients in selecting appropriate refrigeration systems.
